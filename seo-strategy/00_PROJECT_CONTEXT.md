## 1. Was ist MapRaiders?

**Pitch in einem Satz:** GPS-basiertes Outdoor-MMO, das jede Stadt in echte Territorien verwandelt — Spieler erobern reale Karten-Flächen, hinterlassen Standort-Echos und treffen sich bei Live-Events auf der Karte.

**Hauptfunktion**
- GPS-Territorial-Erfassung durch Gehen / Joggen / Radfahren / Auto
- Standort-Echos (Audio, Foto, Video an realen Orten)
- Live-Events auf der Karte (spontan, Push-basiert)
- Verteidigungs-Minispiele (Tic-Tac-Toe, Schere-Stein-Papier, Mini-Schach)
- Clans + Quests + Artefakte
- 100% werbefrei, kein Pay-to-Win, kein Algorithmus-Feed

**USPs (Was uns von Pokémon GO/Ingress/Zenly unterscheidet)**
1. **Echtes Territorium** — keine flüchtigen Gym-Captures, sondern persistenter Land-Besitz
2. **Player-generierter Content** — Echos, Quests, Artefakte werden von Spielern erstellt, nicht von Niantic
3. **Map als Social Network** — kein Algorithmus-Feed, keine Stories, dafür räumliche Nähe als Beziehungs-Anker
4. **No Pay-to-Win** — alle Mechaniken durch Bewegung erreichbar, keine Premium-Items
5. **Werbefrei + Privacy-First** — keine Datenverkäufe, kein User-Tracking-Geschäftsmodell
6. **AR-frei** — kein Akku-killer, kein "AR-Ekel" wie bei Pokémon GO

## 8. Datenschutz pro Markt (2026)

**Wichtig (Korrektur User-Aussage):** DSGVO gilt EU-weit (27 Länder + EEA), nicht nur Deutschland. Andere Märkte haben **eigene Gesetze mit ähnlichen Anforderungen**.

| Markt / Region | Gesetz | Was es für Reviews/Testimonials bedeutet |
|---|---|---|
| **EU/EEA** | **DSGVO/GDPR** | Schriftliche Einwilligung Pflicht, Recht auf Widerruf, bis 4% Jahresumsatz Strafe |
| **UK** | **UK GDPR + DPA 2018** | Praktisch identisch zu DSGVO |
| **Schweiz** | **DSG (revidiert 2023)** | DSGVO-ähnlich |
| **USA-Kalifornien** | **CCPA / CPRA** | Opt-out-Rechte, bis $7,988/Verstoß, GPC-Signal honorieren |
| **USA (20 weitere States bis 2026)** | versch. State Laws | Texas, Virginia, Connecticut, Colorado, Oregon, Delaware u.a. |
| **Brasilien** | **LGPD** (GDPR-modeled, ANPD) | Schriftliche Einwilligung Pflicht |
| **Mexiko** | **LFPDPPP** | Datenschutzhinweis + Einwilligung |
| **Türkei** | **KVKK** | Explizite Einwilligung Pflicht (sehr strikt) |
| **Russland** | **152-FZ** | Datenlokalisierung Pflicht (Daten russischer Bürger müssen in RU gespeichert werden) |
| **Japan** | **APPI** | Einwilligung + Zweckbindung |
| **Süd-Korea** | **PIPA** (eines der striktesten Asien) | Detaillierte Einwilligung pro Datenkategorie |
| **China** | **PIPL** | Explizite Einwilligung, Datenlokalisierung, Behörden-Notifizierung |
| **Indien** | **DPDP Act 2023** | Implementing Rules in Finalisierung |
| **Saudi-Arabien** | **PDPL** (GDPR-aligned) | Einwilligung + Datenexport-Beschränkungen |
| **Indonesien** | **PDP-Gesetz 2022** | DSGVO-ähnlich, Konkretisierung läuft |
| **Australien** | **Privacy Act + APP** | Einwilligung + Notifiable-Data-Breach |
| **Kanada** | **PIPEDA** | Einwilligung + Right of Access |

**Praktische Konsequenz für Tester-Reviews:**
- ✅ **Universell sicher:** Echte Personen, dokumentierte Einwilligung (Email, WhatsApp, Slack reicht), Vorname+Initial, kein Foto, Recht auf Widerruf in Privacy-Policy genannt
- ❌ **Niemals OK:** Fake-Personen, AI-generierte Avatare als "echte Tester" (EU AI Act 2025!), Pseudonyme als "echte Tester" verkaufen, Reviews verfälschen
- ⚠️ **Markt-spezifisch:**
  - **DSGVO:** Aktiv schriftliche Bestätigung mit Recht auf Widerruf
  - **KVKK Türkei:** dito + lokal-arabische Sprache wenn Tester türkisch
  - **PIPL China:** Wenn Tester chinesische Bürger sind → schwierig, lieber meiden
  - **Russland:** Wenn russische Tester → Datenlokalisierung kann Konflikt erzeugen
  - **AI Act EU 2025:** **Markierungspflicht für AI-generierten Content** — falls wir Avatare nutzen würden, müssten sie als "AI-generated" gekennzeichnet sein. Wir nutzen keine Avatare → kein Problem.
