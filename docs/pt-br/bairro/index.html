Perguntas frequentes

É preciso caminhar muito para jogar?

Não. Uma caminhada de 10 minutos pelo bairro é suficiente para conquistar ruas e ganhar recursos.

Posso defender o meu bairro?

Sim. Podes fortalecer territórios, colocar defesas e aliar-te a vizinhos para repelir invasores.

Funciona em zonas rurais?

Absolutamente. Tanto em Lisboa como numa aldeia do Alentejo, cada rua está mapeada e é conquistável.

A app é gratuita?

MapRaiders é completamente gratuito. As funções core de conquista não requerem qualquer pagamento.
