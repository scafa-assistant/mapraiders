FAQ

Common questions

What are spontaneous events?
Spontaneous events are real-world meetups created live on the MapRaiders map. Any player can create an event at their current location — a dog walk, a sports session, a party, a study group — and nearby players see it pulsing on the map and can join in real time.
How do I create an event?
Tap the map at your current location and select 'Create Event'. Give it a name, set a duration, and optionally add a description. The event goes live immediately and appears as a pulsing marker to all players within the discovery radius.
How far away can others see my event?
Events are visible to players within a configurable radius, typically 500 meters to 2 kilometers. Larger events created with special event items can have extended visibility. The radius ensures events stay local and relevant.
How long do events last?
Events are time-limited to keep the map fresh. You set the duration when creating the event, from 30 minutes to a few hours. Once the time expires, the event disappears from the map automatically.
Can I make an event private?
Yes. You can create private events visible only to your friends or clan members. Private events appear with a lock icon and require an invitation or membership to see and join.
What types of events work best?
Any outdoor activity works well. Popular event types include dog walks, jogging groups, skateboarding sessions, outdoor workouts, picnics, photography walks, and territory conquest missions. The app is designed for any reason to get outside.
