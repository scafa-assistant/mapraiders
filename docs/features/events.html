Event-Kategorien

🎮

Kultur & Kunst

Konzerte, Ausstellungen, Strassenkunst, Poetry Slams. Kulturelle Events in deiner Stadt, entdeckt ueber die Karte.

JETZT!

Spontan-Events pulsieren

Wenn ein Event innerhalb der naechsten 24 Stunden startet, beginnt es auf der Karte zu pulsieren. Je naeher der Start, desto staerker der Puls. So siehst du sofort, wo gleich etwas passiert.

Events die JETZT gerade laufen, erhalten ein spezielles Badge. Es leuchtet golden und signalisiert: Hier ist gerade etwas los. Komm vorbei, solange es laeuft.

JETZT!

Event erstellen

In vier einfachen Schritten von der Idee zum Live-Event auf der Karte.

01

Kategorie waehlen

Waehle die passende Kategorie fuer dein Event. Sport, Party, Meetup oder eine andere Aktivitaet.

02

Details eintragen

Titel, Beschreibung, Datum, Uhrzeit und maximale Teilnehmerzahl. So wissen alle Bescheid.

03

Ort festlegen

Waehle den Treffpunkt auf der Karte oder nutze deinen aktuellen Standort. GPS-genau und fuer alle sichtbar.

04

Veroeffentlichen

Dein Event erscheint sofort auf der Karte. Spieler in der Naehe werden benachrichtigt und koennen teilnehmen.

Event-Features

Jedes Event bietet interaktive Funktionen fuer Organisatoren und Teilnehmer.

💬 Event-Chat

Jedes Event hat einen integrierten Gruppenchat. Teilnehmer koennen sich absprechen, Fragen stellen und Updates teilen. Der Chat bleibt aktiv, solange das Event laeuft.

  • Echtzeit-Gruppennachrichten
  • Standort-Sharing im Chat
  • Organisator kann Ankuendigungen pinnen
  • Chat-Verlauf fuer alle sichtbar

📍 GPS-Praesenzcheck

Wenn du am Event-Ort ankommst, kannst du dich per GPS einchecken. Das "Ich bin da!"-Feature bestaetigt deine physische Anwesenheit und zeigt anderen Teilnehmern, wer schon vor Ort ist.

  • Automatische Standort-Verifizierung
  • "Ich bin da!" Button am Event-Ort
  • Anwesenheitsliste in Echtzeit
  • Teilnehmer-Positionen auf der Mini-Map

🔔 Benachrichtigungen

Werde ueber nahende Events in deiner Umgebung informiert. Push-Benachrichtigungen, wenn ein Event bald startet oder wenn neue Events in deinem Gebiet erstellt werden.

  • Push-Alerts fuer Events im Umkreis
  • Erinnerung 30min vor Start
  • Benachrichtigung bei neuen Teilnehmern
  • Absage-Notifications

⭐ Bewertungen

Nach dem Event koennen Teilnehmer den Organisator bewerten. Zuverlaessige Event-Ersteller erhalten einen Vertrauens-Score, der auf ihrem Profil sichtbar ist.

  • Sterne-Bewertung (1-5)
  • Vertrauens-Score auf dem Profil
  • Verifizierte Organisatoren-Badge
  • Event-Historie einsehbar
