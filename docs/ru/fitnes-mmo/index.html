Частые вопросы
Всё что нужно знать
Как фитнес связан с игрой?

Каждый км генерирует очки действий. Больше движения = более мощные территории.

Какой спорт подходит лучше всего?

Бег, ходьба, велосипед — любое движение на улице с активным GPS засчитывается.

Есть ли дуэли между игроками?

Да! Дуэли по дистанции или частоте за неделю против соседних игроков.

Игра бесплатная?

Базовая игра полностью бесплатна. Доступны опциональные косметические элементы.
