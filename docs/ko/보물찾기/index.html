다운로드하고 프로필 생성

숨겨진 미션 찾기

GPS가 특정 장소로 안내——사찰 뒷골목, 전통시장 안, 공원 구석. 현장에서 수수께끼를 풀기.

3

정복하고 흔적 남기기

클리어 장소가 영토로. 에코(음성·사진)를 남겨 다음 탐험가에게 메시지를.

— 베타 테스터, 서울

자주 묻는 질문

포켓몬 GO와 같은 건가요?
아닙니다. 영토 전략, 클랜, 결투가 있는 본격 MMO입니다.
미션은 플레이어가 만드나요?
네. 실제 장소에 미션 생성 가능. 100% 커뮤니티 콘텐츠.
동네를 잘 알아야 하나요?
오히려 반대——몰랐던 골목으로 안내해 줍니다.
혼자 플레이 가능한가요?
네. 클랜에 가입하면 훨씬 풍부한 경험.
