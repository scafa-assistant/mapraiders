다운로드하고 프로필 생성

어디서든 달리기

달리기 전에 MapRaiders를 여세요. GPS가 모든 섹터를 기록합니다 — 한강공원, 올림픽공원, 부산 해운대.

제국 건설

섹터가 쌓입니다. 더 많이 달릴수록 영토가 넓어집니다. 클랜과 협력해 도시 전체를 지배하세요.

자주 묻는 질문

그냥 게임 테마 피트니스 앱 아닌가요?
아닙니다. 영토 경제, 클랜, 결투가 있는 진짜 MMO입니다. 잠든 사이에도 세계가 변합니다.
Strava와 연동되나요?
자체 GPS를 사용하고 GPX를 내보낼 수 있지만 외부 계정은 필요 없습니다.
달리지 않으면 영토가 어떻게 되나요?
서서히 감소합니다. 정기적으로 방문하거나 방어 시설을 건설하세요.
자전거로도 가능한가요?
네. 러너, 사이클리스트, 워커 모두 같은 세계에서 플레이합니다.

당신의 도시를 정복할 준비가 됐나요?

MapRaiders가 곧 Android에 출시됩니다. 지금 등록하고 우선 접근권과 파운더 보너스를 받으세요.
