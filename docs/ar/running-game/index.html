لا. MapRaiders لعبة MMO حقيقية تضم اقتصاداً للأراضي وعشائر ومبارزات وعالماً يتغير أثناء نومك.

تتدهور تدريجياً. زرها بانتظام أو ابنِ منشآت دفاعية.
