常見問題

常見問題

什麼是 MapRaiders?

MapRaiders 是一款面向 Android 的免費戶外社交應用。玩家通過步行、跑步或騎行佔領真實領地,留下基於位置的信息(回聲),並在實時地圖活動中結識其他用戶。

MapRaiders 中的領地如何運作?

玩家通過物理穿越區域來佔領領地。佔領的領地以玩家的顏色顯示在真實地圖上。其他玩家可以通過防禦遊戲——井字棋、石頭剪刀布或迷你西洋棋——發起挑戰。

MapRaiders 中的回聲是什麼?

回聲是留在真實地點的基於位置的信息——音頻錄音、照片或視頻。其他用戶在訪問同一地點時發現它們。

MapRaiders 免費嗎?

是的。MapRaiders 完全免費,沒有廣告和隱藏費用。應用不通過用戶數據或廣告盈利。

MapRaiders 在哪些平臺上可用?

MapRaiders 目前可在 Android 上通過 mapraiders.com 上的 APK 下載使用。iOS 版本正在開發中。

與寶可夢 GO 有何不同?

與寶可夢 GO 不同,MapRaiders 不使用增強現實,也沒有虛擬生物。它專注於真實領地所有權、物理探索和玩家之間的真實面對面相遇。MapRaiders 也完全無廣告。
