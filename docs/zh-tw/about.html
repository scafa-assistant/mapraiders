使命

社交連接屬於街道，而非信息流

大型社交網絡有一個根本性問題：它們為注意力而優化，而非為連接。它們的商業模式是廣告，工具是算法，結果是躲在發光屏幕後面的孤立。

MapRaiders走的是相反的路。社交網絡就是地圖本身。你看到的不是精心策劃的內容——它是真實的。誰在你的城市活躍，誰佔領領土，誰留下回聲，誰創建任務：這一切都發生在外面，在世界上，對同樣在外面的每個人都可見。

沒有任何資料會因為算法想要最大化互動而被推薦給你。你與其他玩家相遇，是因為你們字面意義上在同一個地方。這種連接無法偽造——這正是重點所在。

創立故事

2025年創立——從一個簡單的問題開始

如果地圖就是社交網絡呢？不是一張添加了社交功能的地圖，而是一個以地理位置為每次互動基礎的平臺。

MapRaiders於2025年在Scafa Investments LLC（美國佛羅裡達州卡特勒貝）旗下創立。契機：Ingress或Pokémon GO等現有位置遊戲表明，人們願意為了數字內容在現實世界中行走。但沒有任何一款遊戲邁出下一步——將地圖作為主要社交媒介來對待。

團隊從一個核心理念出發：領土作為個人聲明。當你在自己城市佔據一塊區域時，那是一個聲明。它是可見的。它需要真實的存在。正是在那裡，有著任何算法都無法製造的社交信號。

技術

為真實GPS性能而構建

移動端 — React Native / Expo

MapRaiders應用使用Expo SDK上的React Native開發。全面採用TypeScript，使用Zustand進行狀態管理，採用隱私優先設計的設備端GPS追蹤。沒有用戶明確操作，位置數據不會離開設備。

伺服器 — Node.js + PostgreSQL + PostGIS

後端棧在Node.js、Express和TypeScript上運行。帶PostGIS擴展的PostgreSQL 16實現高性能空間查詢——多邊形重疊、鄰近檢查、實時領土合併操作。

實時 — WebSocket + Redis

所有實時事件（領土攻擊、回聲檢測、事件更新）通過WebSocket連接流動。Redis處理會話管理、速率限制以及伺服器實例之間的pub/sub。

隱私優先——無追蹤商業模式

MapRaiders不為廣告網絡收集數據。GPS數據僅用於遊戲機制。無第三方追蹤，無數據轉售，應用中無廣告SDK。

我們的價值觀

MapRaiders永遠不會成為的東西

沒有算法

你在地圖上看到的是未經過濾的現實。沒有推薦引擎決定你看哪些內容——只有地理和真實人的活動。

永不投放廣告

MapRaiders永遠不會引入廣告支持的模式。未來收入僅來自創作者的可選打賞和提升遊戲體驗的高級功能——而非操縱用戶。

內容所有權歸用戶

任務、回聲、挑戰和領土屬於創建它們的玩家。MapRaiders是平臺，不是內容所有者。你的創作是你的。
