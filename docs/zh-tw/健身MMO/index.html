健身如何與遊戲連接？

每跑1公裡生成行動點數。運動越多，領土越強大。

哪種運動最合適？

跑步、快走、騎車——開啟GPS在戶外進行的任何運動都有效。

有！可以向附近玩家發起距離或周頻率決鬥。

遊戲免費嗎？
