常見問題

如何佔領領地？
打開應用，在地圖上點擊"佔領"。用GPS圍繞目標區域步行或騎車，繪製邊界後確認佔領。
什麼是回聲？
回聲是您在真實地點留下的語音或文字消息。路過該地點的其他玩家可以聽到或閱讀您的回聲。
MapRaiders何時在中國推出？
MapRaiders目前正在開發中。立即註冊，成為北京、上海、成都的首批玩家。
