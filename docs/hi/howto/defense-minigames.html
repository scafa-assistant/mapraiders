Defense Mini-game क्या है?

Defense Mini-game वो lock है जो आप अपनी territory पर लगाते हो। जो भी player आपका territory छीनना चाहे, उसे पहले आपका set किया हुआ game जीतना होगा। 7 games हैं — हर एक की अलग style, अलग difficulty। इन्हें smartly mix करना ही असली defense strategy है।

भारतीय players के लिए एक natural advantage है — शतरंज का invention India में हुआ था! Mini Chess में भारतीय players globally ज़्यादा experienced होते हैं। Geography Quiz में local knowledge बड़ा फर्क डालती है — अपने शहर के landmarks के बारे में questions set करें जो बाहर वाले नहीं जानेंगे।

Defense Setup करने की Step-by-Step Guide

Defense Menu खोलें

Territory tap करें, 'Defense Setup' button press करें। Current defense status और remaining points दिखेंगे।

Game चुनें

High-value territory के लिए Mini Chess या Strategy Puzzle। Low-value के लिए Tic-Tac-Toe काफी है। Points के हिसाब से balance करें।

Difficulty Set करें

Level 1-5। Level 5 पर attacker सिर्फ ~15% बार succeed करता है। Core territories के लिए maximum level recommend।

Traps लगाएं

1-3 traps set करें आसपास। Trap में फंसा attacker reduced time के साथ game खेलता है — success rate गिरती है।

Defense Log Check करें

Regular रूप से देखें कि कौन attack कर रहा है। बार-बार attack होने वाली territory का game type change करें।

अक्सर पूछे जाने वाले सवाल

Defense set न करें तो क्या होगा?

Territory पर 'No Defense' mark लग जाता है और कोई भी बिना game खेले instantly capture कर सकता है। Territory जीतते ही defense लगाएं।

क्या Defense Game बाद में बदल सकते हैं?

हां, कभी भी। लेकिन change के बाद 30 मिनट का "cooldown" period होता है जिसमें defense temporarily weak रहता है। Enemy inactive time में change करें।
