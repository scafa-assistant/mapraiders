私たちについて

私たちはソーシャルネットワーク
そのものである初めての地図を構築しています

アルゴリズムなし。広告フィードなし。有害なレコメンドなし。リアルな人々、リアルな場所、リアルな出会いだけ — 生きた地図の上で可視化されます。

ミッション

ソーシャルなつながりは街路に属するもの、フィードではない

大手ソーシャルネットワークには根本的な問題があります。注目を最大化するために最適化されており、つながりのためではありません。ビジネスモデルは広告、ツールはアルゴリズム、そして結果は光るスクリーンの背後にある孤立です。

MapRaidersは逆のアプローチを取ります。ソーシャルネットワークとは地図そのものです。あなたが見るものはキュレーションされていません — それは現実です。あなたの街で誰が活動しているか、誰が領土を保持しているか、誰がエコーを残しているか、誰がクエストを作成しているか：これらすべては外の世界で起きており、外にいる人全員に見えます。

アルゴリズムがエンゲージメントを最大化したいがためにプロフィールを勧めることはありません。他のプレイヤーと出会うのは、文字通り同じ場所にいるからです。このような形のつながりは偽造できません — それがまさに要点です。

誕生の経緯

2025年設立 — 一つのシンプルな問いから

地図がソーシャルネットワークだったら？ソーシャル機能を追加した地図ではなく、地理的位置がすべてのインタラクションの基盤となるプラットフォームとしての地図。

MapRaidersは2025年、Scafa Investments LLC（米国フロリダ州カトラーベイ）の下で設立されました。きっかけ：IngressやPokémon GOのような既存のロケーションゲームは、人々がデジタルコンテンツのために現実世界を歩き回ることを厭わないことを示しています。しかし、次のステップを踏んだものはありませんでした — 地図を主要なソーシャルメディアとして捉えることです。

チームは一つのコアアイデアから始めました：領土は個人の声明として。あなたが街の一角を保持するとき、それは声明です。それは見えます。本物の存在が必要です。そしてまさにそこに、アルゴリズムが作れないソーシャルシグナルがあります。

テクノロジー

リアルなGPSパフォーマンスのために構築

モバイル — React Native / Expo

MapRaidersアプリはExpo SDK上のReact Nativeで開発されています。全面TypeScript、状態管理にZustand、プライバシーファーストの設計でオンデバイスGPSトラッキング。ユーザーの明示的なアクションなしに位置データがデバイスを離れることはありません。

サーバー — Node.js + PostgreSQL + PostGIS

バックエンドスタックはNode.js、Express、TypeScriptで動作します。PostGIS拡張機能付きPostgreSQL 16により、高パフォーマンスな空間クエリが可能です。ポリゴンの重複、近接チェック、リアルタイムの領土マージ操作を実現します。

リアルタイム — WebSocket + Redis

すべてのライブイベント（領土攻撃、エコー検出、イベント更新）はWebSocket接続を介して流れます。Redisはセッション管理、レート制限、サーバーインスタンス間のpub/subを処理します。

プライバシーファースト — 追跡ビジネスモデルなし

MapRaidersは広告ネットワーク向けにデータを収集しません。GPSデータはゲームメカニクスのみに使用されます。サードパーティトラッキングなし、データの転売なし、アプリに広告SDKなし。

私たちの価値観

MapRaidersが決してならないもの

アルゴリズムなし

地図で見るものはフィルターなしの現実です。どのコンテンツを表示するかを決めるレコメンドエンジンはありません — 地理と実在する人々の活動だけです。

広告は永遠になし

MapRaidersは広告モデルを導入しません。将来の収益は、クリエイターへの任意のチップと、ゲームプレイを操作ではなく強化するプレミアム機能のみから得られます。

コンテンツの所有権はユーザーに

クエスト、エコー、チャレンジ、領土は、それらを作成したプレイヤーのものです。MapRaidersはプラットフォームであり、コンテンツオーナーではありません。あなたの創造物はあなたのものです。

チーム

独立スタジオ — ベンチャーキャピタルなし、妥協なし

MapRaidersはScafa Investments LLCの下で独立した開発スタジオによって構築されています。ベンチャーキャピタルなし、広告モデルを求める外部投資家なし。これにより、次の資金調達ラウンドではなく、ユーザーに利益をもたらす決定を下すことができます。

チームはモバイル開発、バックエンドインフラ、空間データベース、ゲームデザインの専門知識を組み合わせています。急速な成長よりも長期的な品質にフォーカスしています。
