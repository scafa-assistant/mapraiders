よくある質問

これはゲーム風フィットネスアプリですか？
いいえ。MapRaidersは本物のMMO——領地経済、クラン、デュエル、寝ている間に変わり続ける世界があります。
Stravaと連携できますか？
独自GPSを使用しGPXをエクスポートしますが、外部アカウントは不要です。
走らないと領地はどうなりますか？
領地は徐々に減衰します。定期的に訪れるか防衛施設を建設してください。
自転車でも遊べますか？
はい。ランナー、サイクリスト、ウォーカーが同じ世界でプレイします。
