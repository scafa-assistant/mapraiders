フィットネスとゲームはどう繋がりますか？

走ったkmがアクションポイントを生成。多く動くほど領土が強力になります。

どのスポーツが最適ですか？

ランニング、ウォーキング、サイクリング — GPS有効で屋外運動はすべて有効。

プレイヤー間のデュエルはありますか？

はい！近くのプレイヤーに距離または週間頻度のデュエルを挑めます。

ゲームは無料ですか？

基本ゲームは完全無料。オプションのコスメティックアイテムも利用可能。
