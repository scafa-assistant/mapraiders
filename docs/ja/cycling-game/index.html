どんな自転車でも使えますか？

クロスバイク、電動自転車、MTB — GPS有効で屋外走行すべてが対象。

サイクリングロードでプレイできますか？

もちろん！荒川や淀川サイクリングロードは最適です。

プレイヤー間のレースはありますか？

はい、バーチャルレースと週間距離チャレンジが利用可能。

ゲームは無料ですか？

基本ゲームは完全無料。オプションのコスメティックアイテムも利用可能。
