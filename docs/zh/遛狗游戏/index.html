问题所在

遛狗已成例行公事。
它值得更多。

每天两次，同样的路，同样的小区。遛狗是数百万狗主人最固定的日常之一——但往往还是孤独的体验。你可能认识邻居家的狗，却几乎不认识它们的主人。你发现了天坛公园的新步道，却没有人知道。

现有的遛狗应用解决了错误的问题。它们记录步数或显示附近宠物公园，但不建立真正的联结。没有社区实时地图，没有办法听到转角处另一位狗主人留下的消息。

狗主人需要的是一款不只记录、更能连接的社区应用。每次出门都有游戏化的意义。占领领地是玩法的一部分。

狗主人常见问题

每天遛狗可以使用MapRaiders吗？
当然。MapRaiders为短距离规律行走优化，15分钟或1小时都行，每次都能获得领地、积分和社区互动。
天坛公园或朝阳公园能用吗？
有GPS信号的地方都可以。北京天坛公园、朝阳公园、上海世纪公园、深圳莲花山公园——所有GPS可达户外空间均可游戏。
遛狗时需要一直盯着手机吗？
不需要。MapRaiders在后台记录GPS。专心陪伴爱犬，语音回响用耳机听，遛狗后查看结果。
