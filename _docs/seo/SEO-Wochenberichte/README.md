# SEO-Wochenberichte (alle Projekte)

Zentraler Ordner für die wöchentlichen SEO-/GSC-Checks aller Properties.

**Properties:** mapraiders.com · dopaspeak.com · egons.io · ungehoert.music

**Dateibenennung:** `JJJJ-MM-TT_<projekt>.md` je Property, plus `JJJJ-MM-TT_UEBERSICHT_ALLE_PROJEKTE.md` als Portfolio-Snapshot.

**Modus:** Google Search Console wird ausschließlich **gelesen** — keine Validierungen, keine Einreichungen, keine Entfernungen. Schreibende GSC-Aktionen klickt René selbst.

**dopaspeak.com:** wird von einem anderen Agenten betreut. Hier nur Lese-Befund zur Information — keine Datei-/Server-Änderungen.

**Automatisierung:** Geplante Aufgabe `seo-wochenmonitoring-global` (montags), die alle Properties prüft und die Berichte hier ablegt. Die ältere Aufgabe `mapraiders-seo-wochenmonitoring` deckt nur mapraiders ab und kann nach Verifizierung der globalen Aufgabe deaktiviert werden.
